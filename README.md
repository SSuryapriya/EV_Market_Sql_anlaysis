**EV Market Charging Infrastructure Analysis using SQL**

Project Overview
This project analyzes the Global EV Market Charging Infrastructure 2026 dataset using MySQL. The analysis focuses on EV sales, charging stations, battery capacity, vehicle range, market adoption, and government incentives.

## Dataset
- Dataset Name: Global EV Market Charging Infrastructure 2026
- Format: CSV
- Tool Used: MySQL Workbench

## Objectives
- Analyze EV sales across different countries.
- Identify the top EV brands by sales.
- Compare sales by region and vehicle type.
- Analyze charging stations and battery capacity.
- Study market adoption and CO₂ reduction trends.

## SQL Concepts Used
- SELECT
- DISTINCT
- COUNT
- SUM
- AVG
- MIN
- MAX
- GROUP BY
- ORDER BY
- LIMIT

## Key Analysis
- Total number of records
- Country-wise EV sales
- Region-wise EV sales
- Top EV brands
- Vehicle type analysis
- Charging station analysis
- Average battery capacity
- Market adoption rate
- CO₂ reduction analysis

## Tools Used
- MySQL Workbench
- SQL

## Project Outcome
This project demonstrates SQL skills for data analysis by performing data exploration, aggregation, and business insights using real-world EV market data.
